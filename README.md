## 组件

### Button组件

1. 参数支持

| 参数名 | 参数描述 | 参数类型 | 默认值 |
| :----: | :----: | :----: | :----: |
| type | 按钮类型 | string | default |
| plain | 是否是朴素按钮 | boolean | false |
| round | 是否是圆角按钮 | boolean | false |
| circle | 是否是圆形按钮 | boolean | false |
| disabled | 是否禁用按钮 | boolean | false |
| icon | 图标类名 | string | 无 |

2. 事件支持

| 参数名 | 事件描述 |
| :----: | :----: |
| click | 点击事件 |
